# Gender Distribution In the United States House of Representatives 1855 - 2025
## Alex Miramontes
* [Linkedin](https://www.linkedin.com/in/alexmiramontes/)

## Purpose 
This project examines the gender distribution of members of the U.S. House of Representatives from 1855 to 2025. The goal is to understand how the makeup of the House of Representatives has changed over time.

## Research Objectives

* When did the first woman serve in the House from either party?
    
    * When did the first woman serve in the House from the Democrats?
    * When did the first woman serve in the House from the Republicans?  

* What is the number of the most women serving in the House at one time?

    * What is the number of the most women from the Democrat party serving in the House at one time?
    * What is the number of the most women from the Republican party serving in the House at one time?

* What has been the breakdown of the number of men and women from both parties over the last ten years?

    * What has been the breakdown of women from the Democrats over the last ten years?
    * What has been the breakdown of women from the Republicans over the last ten years?
    
* How has the number of women serving in the House trended overall?

    * How has the number of women from the Democrat party serving in the House trended?
    * How has the number of women from the Republican party serving in the House trended?

## Getting started

To replicate this project:

1. Clone the repository:
```
git clone https://github.com/miramontron/us_house_gender_dist.git
```

2. Create and activate a virtual environment, and install required packages from `requirements.txt`:
* Linux/macOS:
```
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
* Windows:
```
python -m venv venv
source venv/Scripts/activate
pip install -r requirements.txt
```
3. To deactivate:

```deactivate```
